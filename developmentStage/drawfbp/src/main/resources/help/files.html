Files

Files (as well as Reports, Legends, Persons, and Enclosures) are used simply for diagramming purposes - they do not cause any code to be generated.

Right clicking on a file brings up a popup menu showing operations that can be performed on it. These are the following:

  • "Edit Description"
  • Toggle Visible/Invisible
  • "Delete"

A File can be connected to any other block type by arrows - see the section on Arrows.
