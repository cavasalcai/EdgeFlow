Persons

These are shown on the diagram as little people!

Persons (as well as Reports, Legends, Files, and Enclosures) are used simply for diagramming purposes - they do not cause any code to be generated.

Right clicking on a Person brings up a popup menu showing operations that can be performed on it.  These are the following:

  • "Edit Item"
  • Toggle Visible/Invisible
  • "Delete"

A Person can be connected by arrows to any block type - see the section on Arrows.
