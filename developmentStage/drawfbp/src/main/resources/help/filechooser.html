Choosing Files

File names are presented after the folders, and are sorted alphabetically independent of case.

If the current language is specified as Java, the JavaFBP jar file and any jar files added using the "File/Add Additional Jar File" function appear at the head of the list of folders, and can be navigated just like any file folder - it is not necessary to expand them.

The buttons on this panel are fairly self-explanatory.

File type selector The button that may be non-obvious is the "ComboBox" in the middle at the bottom (next to "Files of type:"): this toggles between selecting for the file suffixes specified for the "diagram language" and "All" (all suffixes) - just use the mouse to select the desired option, or use the keyboard (tab buttons and up and down arrows).
