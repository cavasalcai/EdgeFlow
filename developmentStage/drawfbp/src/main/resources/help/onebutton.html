One-button Support

DrawFBP can be used with one-button mouse devices in several ways. Clicking on the "Edit" entry at the highest level brings up the following list of functions:

  • "Edit Diagram Description"
  • "New Block"
  • "Block-related Actions"
  • "Arrow-related Actions"

"Block-related Actions" brings up the same functions that right-clicking on a block brings up. However, the block must be selected (it will be coloured yellow) - this can be done by clicking on it.

"Arrow-related Actions" brings up the same functions that right-clicking on an arrow brings up. However, the arrow must be selected (it will be coloured blue) - this can be done by clicking on it.
